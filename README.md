# test-9A

Este es un proyecto de prueba para practicar el uso de **Git** y **GitHub**.  
Se incluyen ejemplos básicos de clonación, commits, push y manejo de ramas.

##  Comandos básicos

### Clonar el repositorio
```bash
git clone https://github.com/BrayanAlv/test-9A.git
cd test-9A
```

### Crear y cambiar a una nueva rama
```bash
git checkout -b nombre-rama
```

### Ver estado de los archivos
```bash
git status
```

### Agregar cambios al staging
```bash
git add .
```

### Hacer commit con mensaje
```bash
git commit -m "Mensaje del commit"
```

### Subir los cambios a GitHub
```bash
git push origin nombre-rama
```

## 📌 Notas
- La rama principal del proyecto es `main`.
- Usa nombres de ramas descriptivos, por ejemplo:
    - `feature/login`
    - `fix/typo-readme`
    - `hotfix/api-error`
